SettiMize is your go-to plugin for advanced WordPress settings, custom fields, and additional optimization features on your dashboard.

== Description ==

**SettiMize** is a versatile WordPress plugin that enhances your WordPress admin experience by incorporating advanced custom settings, fields, and optimization features.

This powerful plugin streamlines your workflow and boosts productivity by adding several essential tools to your admin dashboard.

== Key Features ==

- **Word Count**: Display word counts directly in the post and page list views. This advanced setting lets you quickly assess content length, helping you maintain consistency and meet content standards without opening each post individually.
- **Featured Images**: Display featured images in the post and page list views. This visual enhancement optimizes your WordPress admin by making it easier to identify content at a glance, improving overall usability and management efficiency.

By integrating these advanced custom settings and fields, SettiMize optimizes your WordPress admin experience, making it an indispensable tool for efficient site management.

== Installation ==

1. Upload the `settimize` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'SettiMize' settings page under the 'Settings' menu to configure the plugin.

== Frequently Asked Questions ==

= How do I enable or disable features? =

Go to the 'SettiMize' settings page under the 'Settings' menu. From that field, you can toggle the features On/Off according to your preference.

= Can I customize the settings further? =

The plugin settings are designed to be straightforward. For any specific query, please get in touch with our [support form](https://wordpress.org/support/plugin/settimize/).
